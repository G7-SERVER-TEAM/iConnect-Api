## Description

This is an API for the Find parking application. It has various modules to control the application process.<br>
We hope you will be happy to use this API to work on your task.

## Installation

```bash
$ npm install
```

## Running the app

**User service**

```bash
# development
$ npm run start user

# watch mode
$ npm run start:dev user

# production mode
$ npm run start:prod user
```

**Authentication service**

```bash
# development
$ npm run start auth

# watch mode
$ npm run start:dev auth

# production mode
$ npm run start:prod auth
```
**Area service**

```bash
# development
$ npm run start area

# watch mode
$ npm run start:dev area

# production mode
$ npm run start:prod area
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
